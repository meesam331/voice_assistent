⚡ Groq LinguaFlow: Real-Time Multilingual Voice Assistant
Groq LinguaFlow is a high-performance voice-to-voice translation application. It leverages the speed of the Groq LPU™ Inference Engine to provide near-instant speech transcription, intelligent LLM reasoning, and natural speech synthesis.

🚀 Key Features
Speech-to-Text (STT): Powered by whisper-large-v3-turbo for ultra-fast, accurate transcriptions.

Intelligent Reasoning: Utilizes Llama-3.3-70b for concise, context-aware responses and translations.

Text-to-Speech (TTS): Integrated with Orpheus-v1 for emotional, high-fidelity voice output.

Multilingual Support: Instant translation into Urdu, Hindi, Spanish, French, and Arabic.

Interactive UI: Built with Streamlit, featuring real-time audio recording and auto-play capabilities.

🛠️ Tech Stack
Language: Python

Framework: Streamlit

APIs: Groq Cloud API (Whisper, Llama, Orpheus)

Audio Handling: Audio-recorder-streamlit, Base64 encoding
